# Automated-Reconnaissance-Tool
A browser-based security reconnaissance platform built with Django. Designed to make recon accessible without command-line knowledge.
## Features
- Port scanning & service fingerprinting
- Directory brute-force enumeration
- HTTP security header analysis
- Multi-user authentication & session management
- Scan history storage and report generation (HTML/text)

## Tech Stack
Python · Django · SQLite · Nmap

## How to Run
pip install -r requirements.txt
python manage.py runserver
